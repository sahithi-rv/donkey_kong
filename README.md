### Install in Linux:

Pygame
`sudo apt-get install python-pygame`

### To Launch the game:
`python startgame.py`

### Game Controls
SpaceBar to jump
Right and Left arrows to navigate forward and backward
Up and Down arrows to navigate up and down the ladders
q or x to quit the game.

Hope you enjoy!!
